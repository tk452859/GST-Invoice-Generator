# 🧾 GST Invoice Generator

A free, professional GST-compliant invoice generator built with Python and Streamlit.
Fill in your details, add line items, and download a formatted PDF invoice instantly.

## 🔴 Live Demo
👉 [Try it here](YOUR_STREAMLIT_LINK)

## Features
- CGST + SGST and IGST support
- Add multiple line items with HSN codes
- Live total calculation before generating
- Download as professional PDF instantly
- No signup, no ads, completely free
